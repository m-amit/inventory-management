# inventory-management
Inventory-management without DB and using datastructure

Implement an Inventory Management System. The following features must be implemented
1: Add product
2: Remove product
3: View product details
4: Reporting added/removed products in a given time

To be noted:
1: Use java collections instead of database
2: It must be a spring boot service
3: Exception handling scenarios
4: Swagger documentation for API
